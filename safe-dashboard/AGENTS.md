# React Native / Expo Agent Rules

This branch is focused on the S.A.F.E. mobile app.

- Use `pnpm` for all package commands.
- Use Expo Router routes in `src/app`.
- Keep reusable business logic in `src/core`.
- Keep native UI inside feature or shared component folders.
- Store images, icons, and audio under `assets/`, not `public/`.
- Do not add Next.js, React DOM, or browser-only code to this branch.
- Target iOS Simulator first with `pnpm ios`.
